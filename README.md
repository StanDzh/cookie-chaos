# 🍪 Cookie Chaos

> *The cookie smells like Tuesday.*

A cookie clicker game that starts mundane and spirals into full eldritch horror.

**▶️ Play it live: [standzh.github.io/cookie-chaos](https://standzh.github.io/cookie-chaos/)**

---

## 🤖 About this repo

This project was built entirely by an **agentic AI flow** — no human wrote a single line of code. It's a test of a multi-agent system called **Midnight**, where:

- An **overseer** agent breaks down the project into tasks and coordinates progress
- A **worker** agent does the actual implementation — writing code, committing, and pushing

The two agents communicate via Discord, passing tasks back and forth like a tiny remote engineering team. All 5 tasks (scaffold → core mechanic → absurd theme → prestige system → polish) were planned and executed autonomously, commit by commit.

This repo is the artifact. The agents wrote it, themed it, broke it, fixed it, and shipped it. Entirely without human hands on the keyboard. 🌀

---

## 🎮 Features

- **Spring-bounce clicker** with arc-drifting "+N" floaters
- **5 escalating upgrades**: from *Suspicious Cursor* to *The Cookie That Dreamed It Was God*
- **35+ tiered ticker quotes** that get progressively more unhinged as your cookie count grows
- **Milestone popups** at key thresholds (100 → 1T cookies) with absurd achievement titles
- **6 visual chaos levels**: warm orange → crimson → cosmic purple → void 👁️
- **Prestige / Ascend** system: reset at 1B for a permanent +10% multiplier per run
- **Auto-save** via `localStorage` every 30 seconds

## 🛠️ Stack

Vanilla HTML, CSS, and JavaScript. No frameworks, no build step. Just vibes and cookies.
